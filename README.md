# Eclipse Ace Editor for Eclipse Dirigible

Eclipse Ace Editor Plugin for the [Eclipse Dirigible](https://github.com/eclipse/dirigible) WebIDE v3.x and above.

Build the [Eclipse Dirigible](https://github.com/eclipse/dirigible), deploy on e.g. Tomcat Web Container and go to location:

> http://localhost:8080/services/v3/web/ide-ace/editor.html
